猴子也能看懂的 React 教學 - 1

8月 06, 2016 分類 教學

其實標題打錯了，是猴子才能看懂。
所以看不懂就別怪我囉~ （X

那我們就開始吧~

首先，到 React 官網 下載 React 最新版。
在等待下載的同時來說明一下何為 React。

##什麼是 React？
React 是一個 JS 函式庫，負責產生與管理前端的 UI 。它並不是框架。

##React 特色 – 為什麼我要用它？

  1. 用純 JS 在前端產生 HTML （一般來說是在後端產生 HTML 送到前端）
  2. 使用虛擬 DOM，重繪時效率高
  3. 自定義 Component，方便開發
  4. 父子 Component 可透過 props 和 callback 通訊
  5. 只負責 MVC 的 View 部份，所以不算框架，彈性高
  6. 因為完全是 JS 操作 UI 的關係，使得它可以跟後端分離，達到即時互動的效果

對 React 有沒有比較明白了呢？

為什麼要用 JSX？

前面說過，React 是用純 JS 在前端產生 HTML 的，用原生 JS 寫起來會很麻煩，因此，我們可以使用 JSX，它是 JS 對 XML 的擴充語法，以 XML-like 的語法表達 JS 產生元件的函數，使用的話有助於精簡程式碼，而且對於大多數前端工程師來說這個比較直觀，易於閱讀。
